# Day 3 - HTML Practice

## Overview
This HTML file demonstrates:
- How to set a **favicon** in the browser tab.
- Creating **basic HTML tables** with borders, headers, and rows.

## Features
1. **Favicon**  
   - The page uses a favicon from an external image link.
   - You can see the favicon in the browser tab.

2. **HTML Tables**  
   - **First Table**: Displays a list of people with their name, age, and city.
   - **Second Table**: Shows student names, ages, and marks.
   - Tables are created using `<table>`, `<tr>`, `<th>`, and `<td>` tags with a simple border.

## Preview
Example from the first table:

| Name  | Age | City     |
|-------|-----|----------|
| Alice | 25  | London   |
| Bob   | 30  | New York |
